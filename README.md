# yolo11-object-search-engine

## YOLOv11 Image Search is a Streamlit-based computer vision search engine built on top of YOLOv11m.
# The app allows users to:

 * Run object detection on a directory of images using pretrained yolo11m.pt weights

 * Auto-train the model on your 500 custom images during inference

 * Automatically generate metadata containing classes, bounding boxes, and confidence scores

 * Search images using class names (e.g., apple, bus, bicycle) and optional count filters

 * Support OR / AND search modes to refine results

 * Display matching images along with bounding boxes, labels, and a clean grid UI

 * Export results in JSON format

This tool makes finding specific objects inside large datasets simple, fast, and visually intuitive.
